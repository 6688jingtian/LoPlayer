# LoPlayer
一个简易的播放器，支持vue、react等

![效果图](https://github.com/diy4869/LoPlayer/blob/master/example/02.png)

## 支持的格式
- hls
- dash
- mp4
- ogg
- webm

## 特点
- 快捷键全屏 播放暂停 静音
- 不支持flv(没有为什么，都9102年了，谷歌、微软都准备抛弃了，还支持啥flv)
